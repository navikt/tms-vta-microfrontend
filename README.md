# tms-vta-microfrontend

Veien til arbeid som microfrontend

# Kom i gang

- klon repo
- installer avhengigheter `npm i`
- Start appen i utviklingsmodus lokalt ved å kjøre `npm run dev`
- Appen nås på [http://localhost:3000](http://localhost:3000)
